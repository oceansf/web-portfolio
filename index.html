Portfolio
Productivity Pomodoro Cards
Producivity

Motivate your work/study habits with a visual time-tracker

Pomodoro Cards is a time-tracker that lets users work in 25-minute focus increments followed by a 5-minute break time. However, these time periods can be changed to the user's preference.

What makes this app different is that it displays the amount of study sessions completed in the form of "tomatoes" in order to motivate the user to accomplish more.

  • Styled using Material-UI
  • Neomorphic UI design
  • Features dark-mode
  • Interactive sounds (not compatible on mobile or Safari browsers)
